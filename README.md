Meemo's Naturals — Brand Website
A luxury editorial brochure website for Meemo's Naturals, a natural wellness food brand based in Kitengela, Kenya. Built as a fully static single-page site with zero dependencies — no frameworks, no backend, no database.
The site is inquiry-driven: all product CTAs route visitors directly to WhatsApp or a phone call rather than a checkout flow. Design language draws from high-end consumer wellness brands — editorial serif typography, architectural grid layouts, and a muted natural colour palette.
Stack: Vanilla HTML · CSS · JavaScript
Hosting: Deploy-ready for Netlify, GitHub Pages, or Vercel
Features: Animated ticker, scroll reveal, sticky nav, MVC product table, responsive layout, floating WhatsApp button, pre-filled WhatsApp inquiry links per product
